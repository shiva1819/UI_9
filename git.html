Git Guide

  • Integrate your code with github.
  • create a folder from github repositary.
  • installing git.
  • git commands to push your files to github.
  • git clone url(of your github folder url i.e http:url)
  • touch index.html( to create new files from git)
  • once you added some files in folder.
  • git add . (to add All files)
  • git status (check the status of your files)
  • git commit -m 'your comments here' ( to commit the code);
    if it is asking your credentials please enter your github mail id, and user name.
  • git remote ( to check weather you are connected to remote or not)
  • git push (to push code to your file).
  • and refresh your git hub folder, so you should see you files from local folder.
  • git pull -p (to syncup files from your git folder if any new files are added)
